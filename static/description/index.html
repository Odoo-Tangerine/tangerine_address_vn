Base Vietnam Address

The Base Vietnam Address module is an extension for the Odoo system designed to provide a comprehensive database of addresses in Vietnam. With the integration of the Base Vietnam Address module, Odoo users can access a diverse and detailed collection of addresses at every level, from province/city, district, ward/commune to specific areas. The Base Vietnam Address module not only provides basic address data but can also be customized and expanded to meet the specific needs of each business. This helps streamline workflows and improve efficiency for businesses using Odoo in Vietnam.
